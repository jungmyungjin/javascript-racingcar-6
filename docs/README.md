## 🎮 게임 로직

1. 사용자에게 입력 값을 받는다.
2. 입력한 시도 횟수만큼 자동차를 움직인다.
   1. 자동차가 움직인다.
   2. 결과가 출력된다
3. 최종 우승자를 가린다.
4. 최종 우승자를 출력한다.

## ⚙️ 구현 기능

### 인터페이스

- [INPUT] 사용자에게 N대의 경주할 자동차 이름을 받는다.
- [OUTPUT] 실행 결과를 출력한다.
- [OUTPUT] 최종 우승자를 출력한다.
- [OUTPUT] 예외처리에 메시지를 출력한다.

### **기능 함수**

- 무작위 숫자가 4 이상이면 전진 하고 아니면 멈춘다.
- 경주할 자동차 중 우승자를 찾아 리턴한다.

### **검증**

- 사용자 입력값 검증
  - 사용자 이름 입력 값이 올바른지 검사한다.
  - 시도할 횟수가 1~**`Number.MAX_VALUE`** 인지 검사한다.
